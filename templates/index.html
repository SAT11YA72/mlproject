Student Exam Performance Predictor

About the Project

This project predicts a student's math score based on their personal and educational details. By analyzing features like gender, race/ethnicity, parental education, and test preparation courses, the model provides an accurate math score prediction. The dataset is sourced from Kaggle: Students Performance in Exams Dataset

How It Works

Simply input the student's details such as their gender, ethnicity, and scores in reading and writing. The model will then predict the expected math score based on the provided information.
